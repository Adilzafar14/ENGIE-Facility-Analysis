# 🏢 ENGIE Facility Analytics

> End-to-end data analysis for ENGIE Solutions — 
> Energy Consumption & Predictive Maintenance

## 📊 Projects Overview

### Project 1 — Energy Consumption Analysis
- **Dataset:** UCI Household Power Consumption (2M+ records)
- **Tools:** Python, Pandas, Matplotlib
- **Key Finding:** AC/Heater consumes 72.7% of total energy
- **Insight:** Peak consumption at 8PM — shift loads to 1AM-5AM

### Project 2 — Predictive Maintenance Analytics
- **Dataset:** AI4I 2020 Predictive Maintenance (10,000 records)
- **Tools:** Python, Pandas, Matplotlib
- **Key Finding:** Heat Dissipation Failure = #1 cause (115 cases)
- **Insight:** Replace tools before 130 min to prevent failures

### Project 3 — Master KPI Dashboard
- **Combined:** Both datasets unified
- **Overall Score:** 74% (Target: 75%)
- **Best KPI:** Maintenance Completion — 96%
- **Critical KPI:** Heat Management — 68% (needs attention)

## 💡 Recommendations for ENGIE

1. 🔴 **Install cooling systems** — HDF is #1 failure cause
2. 🔴 **Set tool alerts at 130 min** — failures peak at 143.8 min
3. 🟠 **Upgrade Low Quality machines** — 3.9% failure rate
4. 🟠 **Shift peak loads** to off-peak hours (1AM-5AM)
5. 🟢 **Smart thermostats** — AC/Heater = 72.7% energy use
